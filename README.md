This provides a step-by-step guide to calculate the **work function** and **electrostatic potential** using **VASP (Vienna Ab initio Simulation Package)**.

The work function is an important surface property that determines **electron emission, band alignment, and surface reactivity** in materials such as metals, semiconductors, and 2D systems.

In this tutorial, you will learn how to:

* Set up **surface slab calculations** in VASP
* Create a **vacuum region** for accurate surface modeling
* Extract the **electrostatic potential (LOCPOT)**
* Perform **planar averaging of the potential**
* Identify the **vacuum level and Fermi energy**
* Calculate the **work function (Φ = Evac − Ef)**

This tutorial is useful for **students, researchers, and computational materials scientists** working in **DFT, surface science, and electronic structure calculations**.
